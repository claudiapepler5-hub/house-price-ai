# 🏠 AI House Price Estimator

An interactive web app that estimates home prices based on property features.

## Features
- 6 input factors: square feet, bedrooms, bathrooms, age, location score, garage
- Random Forest machine learning model
- "Surprise Me" button for random examples
- Instant price estimates

## Tech Stack
- Python
- scikit-learn (Random Forest)
- Gradio (web interface)
- - NumPy

## Run It Yourself
1. Open in Google Colab
2. Run the first cell to install Gradio
3. Run the second cell to launch the web app
